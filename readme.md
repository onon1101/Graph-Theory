# Graph-Theory

### demo

迷宮動畫的算法演釋，由入口指向出口的順序。
![](dist/test.gif)
![](dist/test.mp4)

### How to Run
```bash
git clone https://github.com/onon1101/Graph-Theory.git
cd Graph-Theory
pip install -r requirements.txt
python src/App.py
```
### 備註

```bash
pip freeze > requirements.txt

conda env export > enviroment.yml
```
